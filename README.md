# Php Framework 
```
Php Cord operation and model programing framework.
```
## Installation
```
Use the xampp etc. localhost program.
```
## Usage
```
Php-Framework / Models / DbContext.php modify the -> $base_url = "your local url";(C:/xampp/htdocs/php-framework)

Php-Framework / Models / DbContext.php To create a database table use to Start() function.(Suggested usage areas = admin/login.php, base url/index.php)

Php-Framework / Framework / DB.php, modify the connection strings.

Php-Framework / Models / Build the model and database table config you will use.(Sample:User.php)

```
